# Driver-sleep-detection
an smart safety system that detects eye closure using open cv and triggers hardware alerts and telegram notification 

[Youtube Video](https://youtu.be/Ro1PGaYQGAA?si=fOacbK1fvZLjC9Ne) 

# Used 

- Python 3.11
- OpenCV
- MediaPipe
- SciPy

- PySerial

- Arduino UNO

- ESP32

- Telegram Bot API
